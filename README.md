## 프로젝트 소개

### 문제 정의

최근 숏폼 영상의 부상과 빠르게 변화하는 트렌드로 인해, 많은 기업들이 **사용자의 주목을 끌 수 있는 마케팅 애셋을 빠르게, 그리고 적시에 제작**하는 데 어려움을 겪고 있습니다.

Sora, Veo 등 다양한 비디오 생성 모델과 서비스가 등장하고 있지만, 실무 환경에서는 여전히 다음과 같은 제약과 어려움이 존재합니다:

| 제약 사항 | 실무 영향 |
|-----------|-----------|
| **10초 내외의 짧은 영상 길이** | 30초 이상 광고 제작 시 여러 클립 수동 조합 필요 |
| **프롬프트 설계의 어려움** | 원하는 결과물을 얻기 위한 반복적인 시행착오 |
| **목적에 맞는 모델 선택** | Sora vs Veo 등 모델별 특성 파악 및 선택 부담 |
| **일관된 스토리라인 유지** | 여러 클립 간 톤앤매너, 스타일 통일 어려움 |

### 솔루션

OvenAI는 **LLM 기반 멀티에이전트 워크플로**를 통해 이러한 문제를 해결합니다.

에이전트가 사용자의 요청을 바탕으로 **구조화된 역질문을 통해 요구사항을 구체화**하고, 서브 에이전트 단위로 **프롬프트 생성, 이미지·비디오 모델 선택, 결과물 검증**을 수행합니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                         사용자 요청                                  │
│            "쿠키런 캐릭터 변신 광고 30초 만들어줘"                      │
└─────────────────────────────────────────────────────────────────────┘
                                │
                                ▼
┌─────────────────────────────────────────────────────────────────────┐
│                      1. 요구사항 명확화                               │
│  • 타겟 오디언스는?  • 톤앤매너는?  • 필수 포함 장면은?                  │
└─────────────────────────────────────────────────────────────────────┘
                                │
                                ▼
┌─────────────────────────────────────────────────────────────────────┐
│                      2. 스토리보드 생성                               │
│  세그먼트 1 (8초) → 세그먼트 2 (8초) → ... → 세그먼트 N (8초)          │
│  [프롬프트 자동 생성] [모델 자동 선택: Sora/Veo]                        │
└─────────────────────────────────────────────────────────────────────┘
                                │
                                ▼
┌─────────────────────────────────────────────────────────────────────┐
│                      3. 병렬 영상 생성                                │
│  세그먼트별 3개 옵션 동시 생성 (총 21개 영상 ~5분 소요)                  │
└─────────────────────────────────────────────────────────────────────┘
                                │
                                ▼
┌─────────────────────────────────────────────────────────────────────┐
│                      4. 사용자 선택 & 조합                            │
│  각 세그먼트에서 최적 클립 선택 → 최종 영상 자동 조합                    │
└─────────────────────────────────────────────────────────────────────┘
```

### 기대 효과

| 기존 방식 | OvenAI |
|-----------|--------|
| 영상 1개 제작에 **1-2주** | **5분 내 초안**, 1시간 내 완성 |
| 외주 비용 **수백만원/편** | **API 비용만**으로 제작 |
| 기획-제작 간 **다수의 커뮤니케이션** | 에이전트가 **원스톱 처리** |
| A/B 테스트용 버전 제작 **추가 비용** | **세그먼트별 3개 옵션** 기본 제공 |

## 핵심 기능

1. **Clarifying Agent**: 구조화된 역질문으로 사용자 의도 파악 및 요구사항 명확화
2. **Storyboard Agent**: 4-8개 세그먼트로 구성된 스토리보드 자동 생성
3. **Model Selector**: 장면 특성에 맞는 최적 모델 자동 선택 (Sora: 실사/시네마틱, Veo: 스타일라이즈)
4. **Parallel Generation**: 21개 영상 동시 생성으로 대기 시간 최소화
5. **Human-in-the-loop**: 각 단계에서 사용자 승인 및 실시간 수정 가능
